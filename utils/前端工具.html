# 1. responsively (opens new window)

你可以在上面输入你的 web 网站，一次性查看它在各个设备下的响应式布局表现~ 并且支持自定义选择设备、导出为图片等；响应式布局，一个界面尽收眼底！ 👀 再也不用在搁控制台那儿拉来拉去了~~ https://responsively.app/

# 2. figlet

想在控制台上玩点花样，figlet就不容错过了~~ 在项目中 npm install figlet （分清环境：node/Vue/Browser）；

http://www.npmjs.com/package/figlet

# 3.readme.so

顾名思义，readme.so 这个用来快捷生成标准 README 的~

https://readme.so/

# 4.smalldev

这类开发辅助工具还算是挺常见的，不过 smalldev 的优势就是 功能全、颜值高、没广告😎； 代码分享床、JSON 格式化、base64 加解密、HTML/JS/CSS 格式化、压缩 SQL、生成占位图、生成测试数据等等；

https://smalldev.tools/

# 5. smartmockups

据说 smartmockups 是：最好的产品模板生成工具，能创建令人惊艳的产品模板！

https://smartmockups.com/zh

# 6. pfpmaker

https://pfpmaker.com/

# 8. type-scale

type-scale 这个网站用于把不同字体大小，rem 和 em 值的差异进行可视化展现，支持实时预览 https://type-scale.com/ (opens new window)
